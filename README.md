# Beat Maker
A simple web app that allows users to create their own beats. Created to practice JS. 

GH page: https://kayx23.github.io/beat-maker/

### Project Status
Active

### Features
* Sound selection: kick, snare, hihat 
* Tempo adjustment: 20bpm - 300bpm
* Mute tracks
* Animations
